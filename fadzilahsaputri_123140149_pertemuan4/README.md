# Program Pengelolaan Data Nilai Mahasiswa

## Deskripsi Program

Program Pengelolaan Data Nilai Mahasiswa adalah aplikasi berbasis Python yang dirancang untuk membantu mengelola dan menganalisis data nilai mahasiswa secara efisien. Program ini menyediakan berbagai fitur untuk menginput, menampilkan, menganalisis, dan memfilter data nilai mahasiswa dengan antarmuka yang user-friendly.

### Fitur Utama:
-  Menghitung nilai akhir (30% UTS + 40% UAS + 30% Tugas)
-  Menentukan grade (A: ≥80, B: ≥70, C: ≥60, D: ≥50, E: <50)
-  Menampilkan data dalam format tabel
-  Pencarian mahasiswa dengan nilai tertinggi/terendah

### Fitur Tambahan:
- Menginnput data mahasiswa baru
- Memfilter mahasiswa berdasarkan grade
- Menghitung rata-rata nilai kelas

## Cara Menjalankan Program


### Langkah-langkah Instalasi:

1. **Download/Clone Program**
   ```bash
   # Jika menggunakan Git
   git clone <repository-url>
   
   # Atau download file program_nilai_mahasiswa.py secara langsung
   ```

2. **Pastikan Python Terinstall (sesuai instruksi modul)**
   ```bash
   python --version
   # atau
   python3 --version
   ```
   Jika belum terinstall, download dari [python.org](https://www.python.org/downloads/)

3. **Jalankan Program**
   ```bash
   python main.py
   # atau
   python3 main.py
   ```

## Penjelasan Fitur (Konsep Python yang Diimplementasikan)

### 1. Variabel dan Tipe Data
- String (nama, NIM)
- Integer/Float (nilai)
- List (kumpulan data mahasiswa)
- Dictionary (data per mahasiswa)

### 2. Operator
- Aritmatika: `+`, `*`, `/`
- Perbandingan: `>=`, `<=`, `==`
- Assignment: `=`

### 3. Struktur Kendali
- Conditionals: `if`, `elif`, `else`
- Loops: `for`, `while`

### 4. Fungsi
- Fungsi dengan parameter
- Fungsi dengan return value
- Fungsi untuk modularitas kode

### 5. Koleksi Data
- List: Menyimpan multiple mahasiswa
- Dictionary: Menyimpan atribut mahasiswa
- List of Dictionaries: Struktur data utama

### 6. Input/Output
- `input()` untuk menerima data user
- `print()` untuk menampilkan output
- String formatting untuk tampilan rapi

### 7. Error Handling
- `try-except` untuk validasi input
- Validasi rentang nilai
- Pengecekan data kosong
