## Deskripsi
Repository ini berisi hasil pengerjaan **Tugas Praktikum Pemrograman Web RB**  
Seluruh tugas berfokus pada penerapan konsep dasar dan lanjutan dalam pengembangan web, meliputi struktur HTML, styling dengan CSS, interaktivitas menggunakan JavaScript, serta integrasi dengan backend sederhana.

## Tujuan
- Memahami struktur dan sintaks dasar HTML, CSS, dan JavaScript.  
- Menerapkan konsep layout responsif dan desain web modern.  
- Mengembangkan aplikasi web interaktif sesuai studi kasus.  
- Melatih kemampuan kolaborasi dan penggunaan version control (Git & GitHub).

## Tools & Teknologi yang Digunakan
- **HTML5** – Struktur halaman web  
- **CSS3** – Desain dan tampilan web  
- **JavaScript (ES6+)** – Interaktivitas dan logika  
- **Visual Studio Code** – Editor utama  
- **Live Server / Node.js** – Testing & development  
- **Git & GitHub** – Version control dan pengumpulan tugas
